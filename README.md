# IoT Lab 2 - CoreIoT

The project is developed based on multiple steps from the provided tutorials:

* Connecting the device to the CoreIoT platform
* Implementing basic LED blinking functionality
* Controlling NeoPixel LEDs
* Reading temperature and humidity data from the DHT20 sensor
* Sending sensor data to the CoreIoT server

In addition, the system is designed using an RTOS-based approach to handle multiple tasks concurrently.

Beyond the tutorial scope, the system is extended with:

* **WiFi Access Point (AP) mode**, allowing the device to create its own network for direct connection and testing purposes
